Парсер вакансий по навыкам на hh.ru

Информация о парсере


Парсер осуществляет поиск по сайту hh.ru навыков и знаний которыми кандидату необходимо обладать при желании получить ту или иную работу. Например, при поиске по ключевой фразе разработчик python парсер выдаст общее количество вакансий в которых встречается это сочетание и список навыков заявленных работодателем в вакансии.

Поиск всегда осуществляется по ключевому слову или фразе. Навыки выводятся при каждом поиске.

В фильтрах есть дополнительные параметры поиска. Это сферы деятельности и гео регион, по которым сгруппированы все вакансии на hh.ru. При выборе 'искать везде' - поиск будет осуществляться по всему сайту во всех сферах деятельности.

Парсер отсортирует найденые по ключевому слову навыки по нисходящей от более востребованных к менее востребованным.
